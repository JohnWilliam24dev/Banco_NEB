### Cliente:
#### Back end:
- **Criação de conta:**
  - Para criar a conta, é necessário uma struct contendo as informações do cliente/empresa, como por exemplo: Nome, idade, CPF/CNPJ, telefone, endereço, RG, cota salarial, etc.
  - É necessário criar uma senha. Após a senha ser criada, o programa acessará as informações do cliente. O login será o CPF.
  - Durante a criação da senha, censure o que o usuário está escrevendo e faça uma comparação para verificar se as duas senhas são iguais.
  - Implemente um sistema de segurança que não permita que o usuário erre a senha mais de 3 vezes. Caso erre, ele será orientado a procurar um funcionário do banco para resolver o problema.
  
- **Funções disponíveis para clientes:**
  - Verificar o saldo.
  - Realizar transferências, porém apenas se mais de um cliente estiver logado.
  - Mudar senha.
  - Atualizar informações pessoais, exceto RG e CPF.
  - Visualizar extrato de compras (lista de transferências realizadas).
  - Solicitar empréstimo.
  - Fazer um depósito.
  - Após qualquer opção, não retornar ao menu principal.

### Operador:
#### Back end:
- **Criação de conta:**
  - Para criar a conta, é necessário uma struct contendo as informações do funcionário, como: Nome, idade, CPF/CNPJ, telefone, endereço, RG, cota salarial, etc. O Id de funcionário é gerado automaticamente.
  - É necessário criar apenas a senha, pois o login será o ID do funcionário.
  - Implemente um sistema de segurança que não permita que o usuário erre a senha mais de 3 vezes. Caso erre, ele será orientado a procurar um administrador do banco para resolver o problema.

- **Funções disponíveis para funcionários:**
  - Auxiliar clientes em funções que eles tenham dificuldade, necessitando do CPF e senha do usuário.
  - Verificar saldo.
  - Realizar transferências, mas apenas se mais de um cliente estiver logado.
  - Mudar senha do cliente.
  - Atualizar informações pessoais dos clientes, exceto RG e CPF.
  - Visualizar extrato de compras (lista de transferências realizadas).
  - Solicitar empréstimos em nome do cliente.
  - Fazer depósitos.
  - Após qualquer opção, não retornar ao menu principal.
